# Toronto Downtown Parking Survival Guide 2026

A high-end, responsive single-page web application built with Next.js 15, Tailwind CSS, and Framer Motion.

## Features

- **Modern City-Tech aesthetic** with CN Tower Blue (#003399) and Safety Yellow (#FFCC00) accents
- **Dark Mode** toggle (Light / Dark / System preference)
- **Smooth animations** via Framer Motion
- **Interactive comparison grid** for Green P, SpotHero, HonkMobile, and Clicknpark
- **Quick navigation** to Google Maps parking search for Toronto Downtown
- **Expert 2026 tips** in a sticky Pro Tip box

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment (Vercel)

```bash
npm run build
```

Deploy to Vercel with zero config—connect your repo and Vercel will auto-detect Next.js.
